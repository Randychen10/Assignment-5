How to Apply CSS to HTML

To include an external CSS file in an HTML document, use the following example: 

<link rel="stylesheet" href="path/to/your/style.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the following example: 

<script defer src="path/to/your/script.js"></script>
To write JavaScript code within an HTML file, use the following example: 
<script>
      // Your JavaScript code here
    </script>
